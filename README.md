## Description

[Express](https://github.com/expressjs/express) framework TypeScript starter project.

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# Start in development mode
$ pnpm run dev

# Build and run in production
$ pnpm run build
$ pnpm start
```

## Database seed

```bash
$ pnpm seed
```
